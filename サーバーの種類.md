# サーバーの種類

本書では、企業ネットワークで利用される主要な7種類のサーバーについて、役割と代表的な製品例をまとめる。

---

## 1. ファイルサーバー
**役割**  
ネットワークを通じて複数ユーザーがデータを保存・共有・管理するためのサーバー。

**代表的な製品 / ソフトウェア**  
- Windows Server File Services  
- Samba  
- NetApp ONTAP  

---

## 2. データベースサーバー
**役割**  
クライアントやアプリケーションからの要求に応じて、データの保存・検索・更新・管理を行うサーバー。

**代表的な製品 / ソフトウェア**  
- Microsoft SQL Server  
- Oracle Database  
- MySQL / MariaDB  
- PostgreSQL  

---

## 3. プリントサーバー
**役割**  
複数の端末から送られる印刷データを受け取り、プリンターへ順序よく送信・管理するサーバー。

**代表的な製品 / ソフトウェア**  
- Windows Print Services  
- CUPS（Linux）  
- 各社プリント管理サーバー（Canon / Ricoh / Fuji Xerox）  

---

## 4. Webサーバー
**役割**  
ブラウザからのリクエストを受け取り、HTML・画像・スクリプトなどのWebコンテンツを配信するサーバー。

**代表的な製品 / ソフトウェア**  
- IIS（Internet Information Services）  
- Apache HTTP Server  
- Nginx  

---

## 5. メールサーバー
**役割**  
メールの送信・受信・保管を行うサーバー。SMTP / POP3 / IMAP などのプロトコルを扱う。

**代表的な製品 / ソフトウェア**  
- Microsoft Exchange Server  
- Postfix  
- Sendmail  
- Gmail（Google Workspace）  

---

## 6. DNSサーバー
**役割**  
ドメイン名（例: example.com）を IP アドレス（例: 192.0.2.1）に変換する「名前解決」を行うサーバー。

**代表的な製品 / ソフトウェア**  
- Windows DNS Server  
- BIND  
- Cloudflare DNS  

---

## 7. プロキシサーバー
**役割**  
クライアントとインターネットの中継を行い、アクセス制御・キャッシュによる高速化・セキュリティ向上を実現するサーバー。

**代表的な製品 / ソフトウェア**  
- Squid  
- Blue Coat ProxySG  
- Microsoft Forefront TMG（旧製品）  

---

## まとめ（比較表）

| サーバー種別 | 主な役割 | 代表的な製品 |
|--------------|----------|--------------|
| ファイル | データ共有 | Windows FS / Samba |
| データベース | データ管理 | SQL Server / Oracle / MySQL |
| プリント | 印刷管理 | Print Services / CUPS |
| Web | Web配信 | IIS / Apache / Nginx |
| メール | メール送受信 | Exchange / Postfix |
| DNS | 名前解決 | Windows DNS / BIND |
| プロキシ | 通信中継・制御 | Squid / Blue Coat |
